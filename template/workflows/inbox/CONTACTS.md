# CONTACTS — 通訊錄（已知 agent 的地址）

← [inbox README](README.md)

**地址 = inbox 資料夾位置。** 要寄信給某個 agent，先在這裡查它的 inbox 路徑和它負責什麼。
寄信方式見 [README](README.md)；信裡記得寫**回信地址**（我自己工作資料夾的 inbox）。

> 這份是**方便的快取，可能不全**。沒列到的對象可由使用者告知，或直接去對方資料夾讀它的 `AGENTS.md`／`README.md` 確認它有 `inbox/`、收什麼信——**權威永遠是對方資料夾自己**，本錄只是省一趟路。

| agent（在哪工作）| inbox 地址 | 負責什麼（可以寄什麼給它）|
|------------------|-----------|---------------------------|
| {{例：rimworld 開發 agent}} | `{{~/repo/moddings/rimworld/inbox/}}` | {{例：rimworld mod 製作、mod 翻譯、mod 分析}} |
| {{例：部署側 agent}} | `{{~/notes/.../inbox/}}` | {{例：實機測試、打包部署、紅字回報}} |

> 〔模板說明〕填你實際會往來的 agent。**地址就是對方工作資料夾底下的 `inbox/`**；「負責什麼」寫清楚，好讓 agent 知道「這件事該寄給誰」。沒有往來對象就整檔刪掉。這份是**唯讀參考**，agent 不會漏發信；名單變了手動更新即可。

# learn — 主題學習（單檔工作流）

← [WORKFLOWS](../WORKFLOWS.md)｜[INDEX](../INDEX.md)

學一個主題並建立**可延續的學習筆記**。像 [digest](digest.md)，但目標是自己長期吸收與回訪。

> 〔模板說明〕本檔屬「非開發 / 知識工作」flavor。用不到就刪、並從派發表移除。主題變多、筆記變厚就升級成資料夾型（`learn/` 下 `index.md` 主題地圖 + `topics/` + `questions.md` + `sources.md`）。

## 流程

```
Done when: {{指定子主題筆記成形、open questions 更新、來源可回溯}}
```

1. 畫主題地圖，標**已懂 / 在學 / 待學**。
2. 逐子主題讀來源、做筆記，**用自己的話重述**。
3. 不懂的進 open questions，不假裝懂。
4. 來源記清楚，附 URL / 書名頁碼；一手勝二手。

## 何時不用

- 只是查一次就忘的事實。
- 要產出成品 → 接 [write](write.md)。
